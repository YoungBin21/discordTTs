## 사용법
---
문법은 =로 시작한다. <br />
Ex) = 여러분 안녕하세요

봇 내보내기  <br />
== command out

봇은 기본 200초 동안 사용자가 사용을 하지 않으면 자동으로 나갑니다.

현재 태스트 중입으로 실 가용 불가 기술 (command)
사용자가 지정한 특수한 단어를 치면 그에 햐당한 상호작용을 한다.
